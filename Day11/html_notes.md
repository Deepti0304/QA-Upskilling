# Day 11 - HTML, CSS and DOM Notes

## What is HTML?

HTML (HyperText Markup Language) is used to create web pages.

Example:

<html>
<body>
    <h1>Welcome</h1>
    <input type="text">
    <button>Login</button>
</body>
</html>

## Common HTML Tags

- h1 to h6 → Headings
- p → Paragraph
- a → Link
- input → Textbox
- button → Button
- select → Dropdown
- table → Table

## Semantic Tags

- header
- nav
- main
- section
- article
- footer

## Important Attributes

- id
- class
- name
- type
- value
- href
- data-testid

## What is DOM?

DOM (Document Object Model) is a tree representation of HTML.

Selenium interacts with DOM elements.
